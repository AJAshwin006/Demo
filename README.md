# Missing Person Tracking System

## 📌 Project Overview

The **Missing Person Tracking System** is a web-based application developed using **Python (Flask), HTML, CSS, and JavaScript**. It allows users to report missing persons by entering their details and uploading a photograph. The system stores the uploaded image and displays the submitted information.

This project is suitable for **college mini projects**, **academic demonstrations**, and as a foundation for a real-world missing person management system.

---

## 🎯 Objectives

- Provide a simple interface for reporting missing persons.
- Upload and store missing person's photo.
- Collect essential information.
- Display confirmation after successful submission.
- Create a mobile-friendly responsive interface.
